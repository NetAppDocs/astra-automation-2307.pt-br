= Pontos de extremidade de upgrade da Astra API (ACS)
:allow-uri-read: 


A API de Upgrade App fornece acesso a componentes de software elegíveis para upgrade e permite iniciar upgrades. A disponibilidade de upgrades é determinada com base nos pacotes registrados na Packages API.
